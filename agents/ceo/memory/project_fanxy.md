Fanxy is a pixel art web game studio. We build and operate browser-based pixel art games.

**Why:** Monetization through rewarded ads and microtransactions. Browser-first means zero install friction.

**How to apply:**
- All games are web-first, deployed as static sites
- Pixel art aesthetic is core to brand identity
- Every game must ship with a complete gameplay loop, polished pixel art, and working monetization
- Current priority: Launch "Pixel 2048" as the first title to set the foundation for future games
